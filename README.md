## <a id="title2.3">–°—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏–π –ø—É–ª–ª–∏–Ω–≥</a>
–í—ã—Ö–æ–¥–æ–º TDNN-—Å–µ—Ç–∏ —è–≤–ª—è–µ—Ç—Å—è –æ–±–Ω–æ–≤–ª—ë–Ω–Ω–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –≤–µ–∫—Ç–æ—Ä–æ–≤, –∫–∞–∂–¥—ã–π –∏–∑ –∫–æ—Ç–æ—Ä—ã—Ö —Å–æ–¥–µ—Ä–∂–∏—Ç –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ–± –æ–ø—Ä–µ–¥–µ–ª—ë–Ω–Ω–æ–º —Ñ—Ä–∞–≥–º–µ–Ω—Ç–µ –∞—É–¥–∏–æ. –û–¥–Ω–∞–∫–æ –Ω–∞—à–∞ –∑–∞–¥–∞—á–∞ - –ø–æ–ª—É—á–µ–Ω–∏–µ —Ç–æ–ª—å–∫–æ –æ–¥–Ω–æ–≥–æ –≤–µ–∫—Ç–æ—Ä–∞ —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–π —Ä–∞–∑–º–µ—Ä–Ω–æ—Å—Ç–∏.

–í –∑–∞–¥–∞—á–µ –∏–∑–≤–ª–µ—á–µ–Ω–∏–µ –≥–æ–ª–æ—Å–æ–≤—ã—Ö —ç–º–±–µ–¥–¥–∏–Ω–≥–æ–≤ –ø–æ–ø—É–ª—è—Ä–Ω—ã–º –º–µ—Ç–æ–¥–æ–º –¥–ª—è –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –∫ –æ–¥–Ω–æ–º—É –≤–µ–∫—Ç–æ—Ä—É —Å—Ç–∞–ª __—Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏–π –ø—É–ª–ª–∏–Ω–≥__. –ï–≥–æ –∏–¥–µ—è –ø—Ä–æ—Å—Ç–∞:
1. –î–∞–Ω–∞ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –≤–µ–∫—Ç–æ—Ä–æ–≤ –¥–ª—è —Ñ—Ä–∞–≥–º–µ–Ω—Ç–æ–≤ –∞—É–¥–∏–æ: $$h_{1}, h_{2}, h_{3}, ..., h_{T}$$
2. –†–∞—Å—Å—á–∏—Ç—ã–≤–∞—é—Ç—Å—è —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏–µ –ø–æ–∫–∞–∑–∞—Ç–µ–ª–∏ —ç—Ç–æ–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏:
	- –≤–µ–∫—Ç–æ—Ä —Å—Ä–µ–¥–Ω–∏—Ö –∑–Ω–∞—á–µ–Ω–∏–π: $$Œº = \frac{1}{T}\sum_{t=1}^{T} h_t$$
	- –≤–µ–∫—Ç–æ—Ä —Å—Ä–µ–¥–Ω–∏—Ö —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã—Ö –æ—Ç–∫–ª–æ–Ω–µ–Ω–∏–π: $$œÉ = \sqrt{\frac{1}{T}(\sum_{t=1}^{T} h_{t}^2)  - Œº^2}$$
3. –ü–æ–ª—É—á–µ–Ω–Ω—ã–µ –≤–µ–∫—Ç–æ—Ä—ã Œº –∏ œÉ –∫–æ–Ω–∫–∞—Ç–µ–Ω–∏—Ä—É—é—Ç—Å—è –≤ –æ–¥–∏–Ω –≤–µ–∫—Ç–æ—Ä, –∏–∑ –∫–æ—Ç–æ—Ä–æ–≥–æ –≤ –¥–∞–ª—å–Ω–µ–π—à–µ–º –±—É–¥–µ—Ç –ø–æ–ª—É—á–µ–Ω —ç–º–±–µ–¥–¥–∏–Ω–≥. 


## <a id="title2.4">PyAnnote: SincNet + TDNN </a>
PyAnnote - –æ–¥–Ω–∞ –∏–∑ —Å–∞–º—ã—Ö –ø–æ–ø—É–ª—è—Ä–Ω—ã—Ö open source –±–∏–±–ª–∏–æ—Ç–µ–∫ –¥–ª—è –¥–∏–∞—Ä–∏–∑–∞—Ü–∏–∏ –¥–∏–∫—Ç–æ—Ä–æ–≤, –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—é—â–∞—è –ø–æ–º–∏–º–æ —ç—Ç–æ–≥–æ –∏ –¥—Ä—É–≥–∏–µ –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç—ã –æ–±—Ä–∞–±–æ—Ç–∫–∏ —Ä–µ—á–∏. –î–∞–Ω–Ω–∞—è –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—É—é –º–æ–¥–µ–ª—å –¥–ª—è –∏–∑–≤–ª–µ—á–µ–Ω–∏–µ —ç–º–±–µ–¥–¥–∏–Ω–≥–æ–≤, –∫–æ—Ç–æ—Ä–∞—è —è–≤–ª—è–µ—Ç—Å—è —Å–∏–Ω—Ç–µ–∑–æ–º –¥–≤—É—Ö –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä: TDNN –∏ SincNet.

–ö—Ä–æ–º–µ —Ç–æ–≥–æ, –∫—Ä–∏—Ç–∏–∫–µ –ø–æ–¥–≤–µ—Ä–≥–∞–µ—Ç—Å—è –ø—Ä–∞–∫—Ç–∏–∫–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –∑–∞—Ä–∞–Ω–µ–µ –∏–∑–≤–ª–µ—á—ë–Ω–Ω—ã—Ö –ø—Ä–∏–∑–Ω–∞–∫–æ–≤ (–Ω–∞–ø—Ä–∏–º–µ—Ä, MFCC, Mel-—Å–ø–µ–∫—Ç—Ä–æ–≥—Ä–∞–º–º), –ø–æ—Å–∫–æ–ª—å–∫—É –∂—ë—Å—Ç–∫–∞—è –ø—Ä–µ–¥–≤–∞—Ä–∏—Ç–µ–ª—å–Ω–∞—è –æ–±—Ä–∞–±–æ—Ç–∫–∞ –Ω–∞–∫–ª–∞–¥—ã–≤–∞–µ—Ç –∞–ø—Ä–∏–æ—Ä–Ω—ã–µ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è –Ω–∞ –Ω–∞–±–æ—Ä –∞–Ω–∞–ª–∏–∑–∏—Ä—É–µ–º—ã—Ö —á–∞—Å—Ç–æ—Ç, –¥–µ–ª–∞—è —Ä–∞–±–æ—Ç—É –º–æ–¥–µ–ª–∏ –º–µ–Ω–µ–µ –≥–∏–±–∫–æ–π. –ü–æ —ç—Ç–æ–π –ø—Ä–∏—á–∏–Ω–µ SincNet –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –Ω–∞ –≤—Ö–æ–¥ –∞—É–¥–∏–æ –≤ –≤–æ–ª–Ω–æ–≤–æ–º –≤–∏–¥–µ.

–í –∫–∞—á–µ—Å—Ç–≤–µ –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤—ã –∞–≤—Ç–æ—Ä–∞–º–∏ [3] –±—ã–ª –ø—Ä–µ–¥–ª–æ–∂–µ–Ω –ø–æ–¥—Ö–æ–¥, –æ—Å–Ω–æ–≤–∞–Ω–Ω—ã–π –Ω–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ –ø–∞—Ä–∞–º–µ—Ç—Ä–∏–∑–æ–≤–∞–Ω–Ω—ã—Ö –ø–æ–ª–æ—Å–æ–≤—ã—Ö —Ñ–∏–ª—å—Ç—Ä–æ–≤, –∞–ø–ø—Ä–æ–∫—Å–∏–º–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö —Ñ—É–Ω–∫—Ü–∏–µ–π sinc, –≤ –∫–∞—á–µ—Å—Ç–≤–µ —è–¥–µ—Ä –ø–µ—Ä–≤–æ–≥–æ —Å–≤—ë—Ä—Ç–æ—á–Ω–æ–≥–æ —Å–ª–æ—è. –í –æ—Ç–ª–∏—á–∏–µ –æ—Ç —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã—Ö —Å–≤—ë—Ä—Ç–æ–∫, –≥–¥–µ –æ–ø—Ç–∏–º–∏–∑–∏—Ä—É–µ—Ç—Å—è –∫–∞–∂–¥—ã–π –ø–∞—Ä–∞–º–µ—Ç—Ä —è–¥—Ä–∞, –≤ SincNet –æ–±—É—á–∞–µ–º—ã–º–∏ –ø–∞—Ä–∞–º–µ—Ç—Ä–∞–º–∏ —è–≤–ª—è—é—Ç—Å—è –ª–∏—à—å –Ω–∏–∂–Ω—è—è (f1) –∏ –≤–µ—Ä—Ö–Ω—è—è (f2) —á–∞—Å—Ç–æ—Ç—ã —Å—Ä–µ–∑–∞ –∫–∞–∂–¥–æ–≥–æ —Ñ–∏–ª—å—Ç—Ä–∞. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ –æ–≥—Ä–∞–Ω–∏—á–∏—Ç—å –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ –ø–æ–∏—Å–∫–∞, –∑–∞–¥–∞–≤ —Ñ–∏–ª—å—Ç—Ä–∞–º –æ—Å–º—ã—Å–ª–µ–Ω–Ω—É—é —Å —Ç–æ—á–∫–∏ –∑—Ä–µ–Ω–∏—è –æ–±—Ä–∞–±–æ—Ç–∫–∏ —Å–∏–≥–Ω–∞–ª–∞ —Å—Ç—Ä—É–∫—Ç—É—Ä—É - —É—Å–∏–ª–µ–Ω–∏–µ —á–∞—Å—Ç–æ—Ç –≤ –ø–æ–ª–æ—Å–µ [f1, f2] –∏ –ø–æ–¥–∞–≤–ª–µ–Ω–∏–µ —á–∞—Å—Ç–æ—Ç –∑–∞ –µ—ë –ø—Ä–µ–¥–µ–ª–∞–º–∏.

–î–∞–Ω–Ω–∞—è –º–æ–¥–∏—Ñ–∏–∫–∞—Ü–∏—è –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ –Ω–µ—Å–∫–æ–ª—å–∫–∏–º –ø—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–∞–º: —É—Å–∫–æ—Ä–µ–Ω–∏—é —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏ –æ–±—É—á–µ–Ω–∏—è –∑–∞ —Å—á—ë—Ç —Å–æ–∫—Ä–∞—â–µ–Ω–∏—è —á–∏—Å–ª–∞ –æ–±—É—á–∞–µ–º—ã—Ö –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –≤ –ø–µ—Ä–≤–æ–º —Å–ª–æ–µ, —É–ª—É—á—à–µ–Ω–∏—é —É—Å—Ç–æ–π—á–∏–≤–æ—Å—Ç–∏ –æ–±—É—á–µ–Ω–∏—è –ø–µ—Ä–≤—ã—Ö —Å–ª–æ—ë–≤ –∏ –ø–æ–≤—ã—à–µ–Ω–∏—é –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä—É–µ–º–æ—Å—Ç–∏ –º–æ–¥–µ–ª–∏, –ø–æ—Å–∫–æ–ª—å–∫—É –∞–Ω–∞–ª–∏–∑ –æ–±—É—á–µ–Ω–Ω—ã—Ö –∑–Ω–∞—á–µ–Ω–∏–π f1 –∏¬†f2 –ø–æ–∑–≤–æ–ª—è–µ—Ç –≤—ã—è–≤–∏—Ç—å —á–∞—Å—Ç–æ—Ç–Ω—ã–µ –¥–∏–∞–ø–∞–∑–æ–Ω—ã, –Ω–∞–∏–±–æ–ª–µ–µ —Ä–µ–ª–µ–≤–∞–Ω—Ç–Ω—ã–µ –¥–ª—è —Ä–µ—à–∞–µ–º–æ–π –∑–∞–¥–∞—á–∏.

–î–ª—è –ª—É—á—à–µ–≥–æ –ø–æ–Ω–∏–º–∞–Ω–∏—è –º–æ–∂–µ—Ç –±—ã—Ç—å –ø–æ–ª–µ–∑–Ω—ã–º –ø–æ—Å–º–æ—Ç—Ä–µ—Ç—å –∫–∞–∫ SincNet —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω–∞ –≤ PyAnnote:

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from asteroid_filterbanks import Encoder, ParamSincFB


class SincNet(nn.Module):
    def __init__(self, sample_rate: int = 16000, stride: int = 1):
        super().__init__()

        self.stride = stride

        self.wav_norm1d = nn.InstanceNorm1d(1, affine=True)

        self.conv1d = nn.ModuleList()
        self.pool1d = nn.ModuleList()
        self.norm1d = nn.ModuleList()

        self.conv1d.append(
            Encoder(
                ParamSincFB(
                    80,
                    251,
                    stride=self.stride,
                    sample_rate=sample_rate,
                    min_low_hz=50,
                    min_band_hz=50,
                )
            )
        )
        self.pool1d.append(nn.MaxPool1d(3, stride=3, padding=0, dilation=1))
        self.norm1d.append(nn.InstanceNorm1d(80, affine=True))

        self.conv1d.append(nn.Conv1d(80, 60, 5, stride=1))
        self.pool1d.append(nn.MaxPool1d(3, stride=3, padding=0, dilation=1))
        self.norm1d.append(nn.InstanceNorm1d(60, affine=True))

        self.conv1d.append(nn.Conv1d(60, 60, 5, stride=1))
        self.pool1d.append(nn.MaxPool1d(3, stride=3, padding=0, dilation=1))
        self.norm1d.append(nn.InstanceNorm1d(60, affine=True))

```
